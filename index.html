App's Features

List of features that might help you be more productive and effective.

  • Always On Top

    If enabled, the app will always on top of other apps running on user's Operating System.

  • Full Screen Breaks

    If enabled, the app will interrupt the user by occupying the whole screen so that the user will be force to take a break.

  • Special Breaks

    If enabled, the user will be allowed to set some special time when the user really need to take a break.

  • Strict Mode

    If enabled, the app will strictly follow user's configuration and prevent the user from resetting, pausing and skipping the timer when it has started.

  • Dark Theme

    Allowing the user to use Dark Mode to reduce eye strain and improves visibility for user with low vision and those who are sensitive to bright light.

How It Works

You can start by following the basic steps provided bellow.

  1. Create task lists

    List down all the task you want to accomplished using the Built-in Task List.

  2. Choose a task item

    Choose a task item to be accomplished first and put it on the top of your priority list.

  3. Start to focus on work

    Stay focus and work on the task until the Pomodoro ends.

  4. Take a short break

    Do something else relaxing (i.e., not work-related). Your brain will thank you later.

  5. Repeat the process

    Start again. Just repeat the process until you become comfortable of it.

  6. Take a long break

    After your fourth session take a longer break and come back completely refreshed.
